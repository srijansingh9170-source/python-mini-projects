Python Mini Projects

Description
This repository contains a collection of small Python projects created for learning and practicing programming concepts. These projects help beginners understand Python logic, control flow, and problem solving.

Projects Included

- Calculator
- Number Guessing Game
- Password Generator

Features

- Beginner friendly Python programs
- Easy to understand code
- Good practice for Python basics

Installation

1. Clone the repository

git clone https://github.com/srijansingh9170-source/python-mini-projects.git

2. Move to the project folder

cd python-mini-projects

Usage

Run any Python file using:

python filename.py

Example:

python calculator.py
